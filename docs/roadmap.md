# Zuhn — Deferred Features Roadmap

Last updated: 2026-03-26

## Empirical Engine Enhancements

Deferred from Phase 15 resolve pipeline build (2026-03-22). Build these when the system has 5+ resolved predictions/decisions.

### Scoring Refinements
- **Stakes weighting on decisions** — add a `stakes` field (low/medium/high/critical) to DecisionSchema. Higher-stakes outcomes should cascade confidence adjustments with greater magnitude. A successful $100K investment decision carries more empirical weight than choosing a newsletter.
- **Numerical confidence scoring** — migrate from enum (pending/low/medium/high/very_high) to a composite 0-100 score with four components: structural (source corroboration), empirical (outcome track record), temporal (old confirmations decay), diversity (unique perspectives supporting the belief). The enum label maps from the number.

### Tracking & Views
- **Calibration tracking (personal Brier score)** — track prediction accuracy per domain. If Jin marks 70% of predictions as high confidence, he should be right ~70% of the time. Surface over/underconfidence in the morning briefing.
- **Belief portfolio view** — `views/belief-portfolio.md` showing battle-tested principles (many confirmed predictions), untested principles (no predictions derived), and shaky principles (failed predictions). Includes calibration summary.

### Active Intelligence
- **Bounty system** — four bounty types generated by learn.ts: gap fill, contrarian challenge, perspective blind spot, staleness alert. Surfaced in morning briefing as opt-in search queries.
- **Adversarial bounties** — for every principle with confidence > high, generate a search for counterarguments. Track source diversity per principle.
- **Decision playback** — when resolving a decision, generate a retrospective: what you believed, what happened, what the KB would say if you asked today.

### Phase 21: Source Quality Scoring
- Weight insights by source credibility — Sequoia Capital, peer-reviewed research, and domain experts carry more authority than clickbait YouTubers
- Source reputation score (0-100) based on: citation count, credentials, prediction track record, peer corroboration
- Propagate source quality into insight confidence — an insight supported only by low-quality sources should have dampened confidence
- Surface in views: which principles rest on strong vs weak evidentiary foundations
