**Side A:** "Domain experts build better vertical software than software-first developers once vibe coding makes code cheap."
**Side B:** "Vertical domain expertise only becomes a durable moat when paired with accountable software operations."
**Reason:** The first stance treats domain understanding as the binding constraint after AI collapses implementation cost. The GitLab AI Accountability evidence pressures that by showing code generation speed creates downstream traceability, governance, review, security, compliance, deployment, and maintenance bottlenecks. The contradiction is conditional rather than total: domain experts may build better concepts, but production advantage may require technical accountability infrastructure rather than domain expertise alone.
