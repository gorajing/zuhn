**Side A:** "Reinforcement learning plus inference-time compute scaling unlocked tool use and agentic coding as a genuine capability breakthrough"
**Side B:** "While per-token costs dropped, reasoning models burn so many more tokens per useful output that actual inference costs per task have increased"
**Reason:** The capabilities are real and the cost increase is real. Is inference-time compute a breakthrough (new capabilities justify higher costs) or an economic trap (capabilities that cost more than the value they create)?
**Resolution:** Synthesis — Inference-time compute is a genuine breakthrough AND an economic trap — for different task tiers. Breakthrough for the $50-500/task tier (agentic coding, research). Economic trap for the $0.01-1.00/task tier (consumer queries). The error is treating inference compute as a single cost category.
