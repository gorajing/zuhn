# Tag: golden-dataset

- `INS-260626-AC18` When you have no production traffic, curate known edge cases into a golden dataset so releases rest on evidence rather than vibes.
- `INS-260605-8789` Two experts grading the same output against the same rubric often agree only 20-30% of the time, so a judge hitting 0.4 agreement with you is doing 'really, really well.'
- `INS-260605-C009` LLM-as-judge is 'the lunatics running the asylum' — write your own deterministic eval against a golden set when you can.
- `INS-260605-5D14` If the user replies 'no, I meant X,' the agent failed; if they say thanks and leave, it worked — that beats any thumbs button.
- `INS-260628-0AC7` A fast eval is a set of query-document pairs you measure by checking whether the right docs come back in the top-k, and it should run in seconds for pennies.
- `INS-260628-7A77` Scoring 100% on a finite golden set tells you nothing about the inputs 'around the corner' that you never tested.
