# Tag: gene-editing

- `INS-260323-3657` CRISPR-Cas9 uses RNA-guided proteins to make targeted DNA edits cheaply and quickly, unlike earlier one-off gene-editing tools.
- `INS-260323-D4B1` The same gene editing tools used for de-extinction can save living species from genetic bottlenecks, invasive species, and disease — a far more immediate and impactful application than bringing back mammoths.
- `INS-260323-8CA9` Scientists pursuing de-extinction deliberately make the fewest genetic edits possible, because each change carries unpredictable risk — a principle applicable to any complex system intervention.
- `INS-260323-133A` Editing your own cells (somatic) affects only you; editing embryos (germline) permanently changes all future descendants — a fundamentally different ethical calculus.
