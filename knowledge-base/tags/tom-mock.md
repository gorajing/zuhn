- `INS-260514-0239` Game selection beats game excellence. Hedge funds, IB, big tech — crowded high-status games. Ambulance supply, specialty B2B, niche distribution — uncrowded with similar/better outcomes. Choose the game, then play it well.
- `INS-260514-B7CA` Allocators talk to each other and can deconstruct your portfolio from returns alone. Be ruthlessly honest about wins AND losses. Lying about returns gets you blacklisted across the entire allocator network — not just one firm.

- `INS-260514-8BD5` Markets aren't random — they're deterministic competition between rational agents trying to outsmart each other. Every inefficiency exists because someone else thinks they'll profit. Model their incentives, don't time the market.
