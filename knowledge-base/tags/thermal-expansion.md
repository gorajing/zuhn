# Tag: thermal-expansion

- `INS-260330-FD7C` Picosecond lasers shatter tattoo ink because the outer layer of each particle expands thermally before heat reaches the core, creating mechanical stress that fractures the particle.
- `INS-260330-F76B` Picosecond laser pulses heat one side of an ink particle so fast that thermal expansion on that side rips it apart before the other side warms up.
