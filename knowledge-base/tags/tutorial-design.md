# Tag: tutorial-design

- `INS-260329-18BB` Marathon-format tutorials exploit commitment psychology — once a learner invests 30+ minutes, sunk cost and flow state reduce the probability of abandonment compared to a series where each video is a natural exit point.
- `INS-260329-BCCB` Long-form tutorials that build cumulatively create stronger foundational understanding than consuming the same material in disconnected short videos.
- `INS-260329-E20C` The ~2 hour tutorial length balances completeness with finishability, letting beginners go from zero to a working result in one session.
