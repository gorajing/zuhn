Graham articulates a subtle compounding mechanism: when founders improve at choosing which problems to work on, they benefit twice — the path to success becomes shorter (fewer wrong turns), and they can move faster along the right path because they're more certain it's correct. This is analogous to a running back's downfield path — winding but executed rapidly.

The practical framework is to operate with high decisiveness at the micro scale (weekly action cycles with measurable results) while remaining tentative at the macro scale (willing to course-correct). This creates a rapid zigzag that converges faster than either pure caution or reckless speed. The key enabler is fast feedback loops — trying ideas, measuring results, and adjusting within a week or less.
