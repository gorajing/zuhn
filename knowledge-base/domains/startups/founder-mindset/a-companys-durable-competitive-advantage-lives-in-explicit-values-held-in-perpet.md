The 2025 letter is unusual in that it devotes substantial space to explicit value articulation rather than just business results. Greg writes: 'These foundational values built Berkshire, and equip us to succeed in the decades ahead. While we have set them out explicitly this year, we will publish them as an attachment to future letters, with each letter discussing how we practiced those values across Berkshire.' The mechanism is: turn implicit founder wisdom into explicit operating doctrine so the next generation of managers can execute the same playbook without access to the founder. This is the CODE equivalent of moving from tacit apprenticeship to an executable manual. Most companies either don't do this (and lose identity after the founder leaves) or do it too late (after identity has already drifted). Doing it during the successor's first letter, while the founder is still alive to validate it, is the unusual move.
