The emergence of industrial capitalism in Britain demonstrates how positive feedback loops drive exponential system growth. When agricultural innovations (crop rotation, nitrogen-fixing clover) lowered food prices, workers retained more disposable income. This surplus income created demand for manufactured consumer goods, which incentivized entrepreneurs to invest in more efficient production methods, further lowering prices and expanding markets.

This pattern is directly applicable to modern startups: the most durable businesses create feedback loops where improving one variable (cost, quality, speed) naturally drives improvement in others. Platforms that reduce marginal costs while expanding demand — like AWS reducing compute costs which drives more startups which drives more AWS usage — mirror this 16th-century agricultural-to-industrial transition. The key insight is that these loops are not designed top-down but emerge from structural conditions that align incentives across producers and consumers.
