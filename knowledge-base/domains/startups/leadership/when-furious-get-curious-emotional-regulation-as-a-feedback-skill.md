Scott's framework provides a practical gauge for reading feedback reactions. If someone seems sad, it signals you need to demonstrate more care. If someone seems angry, the instinct is to get defensive or back off — but both are wrong. Instead, 'get curious, not furious' — ask what's driving the anger.

This is critical because the natural overcorrection from obnoxious aggression (challenging without caring) isn't to care more, but to stop challenging — which dumps you into the worst quadrant, manipulative insincerity. The only productive path from any failure mode is up and to the right: more care and more challenge simultaneously.
