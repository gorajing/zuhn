RuBisCo is the most abundant protein on Earth (~40 billion tons at any time) despite being so inefficient it confuses oxygen for CO2 roughly half the time. Plants couldn't switch to a better enzyme because their entire metabolic architecture was built around it — so they evolved workarounds for the toxic byproducts instead.

This maps directly to startup dynamics: the first solution to achieve ecosystem lock-in often wins permanently, even if technically inferior. Competitors building 'better' alternatives face the same challenge — the incumbent's entire value chain has co-evolved around its specific quirks. The strategic response isn't to build a perfect replacement but to either work within the existing architecture or find an entirely new niche where the incumbent's path dependencies become irrelevant.
