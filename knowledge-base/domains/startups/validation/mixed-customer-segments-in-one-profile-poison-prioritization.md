When companies group jobs, pains, and gains from multiple customer segments (e.g., payers and end users) into one canvas, they create an averaged profile that accurately represents no one. Prioritization becomes impossible because high-priority needs for one segment get diluted by irrelevant needs from another.

The fix is mechanically simple: create separate canvases for each segment, then design targeted value propositions for each. This also reveals where segment needs conflict — a payer might prioritize cost reduction while a user prioritizes experience — forcing explicit strategic choices about which segment to optimize for.
