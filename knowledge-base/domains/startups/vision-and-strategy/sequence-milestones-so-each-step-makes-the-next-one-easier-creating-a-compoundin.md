SpaceX's compounding loop is: reusability lowers cost → lower cost increases launch demand → more launches improve reliability → reliability unlocks new markets. Each step is both an outcome and an input to the next. This is qualitatively different from a milestone list where items are independent checkboxes.

The strategic value is twofold. First, it makes long-term vision feel grounded rather than speculative—Mars doesn't feel like science fiction because it sits at the end of a chain of causally connected steps. Second, it gives investors a falsifiable signal: if step N fails, they know which downstream bets to revise. This is the kind of structured uncertainty that sophisticated investors can reason about.

For founders building a roadmap: draw the causal arrows between milestones explicitly. If you can't articulate how winning at step 2 makes step 3 cheaper or more likely, the roadmap is a timeline, not a strategy.
