Bing is widely mocked as inferior to Google, yet it generates roughly $5 billion per year in profit. The entire explanation is distribution: Bing is the default search engine on every Windows computer and Internet Explorer/Edge browser. Most users never change defaults — not because they prefer Bing, but because changing requires deliberate effort most people won't take.

This has profound implications for startup strategy. Competing on product quality alone is insufficient when an incumbent owns the default position on a platform layer above you. The strategic priority should be securing default distribution channels before optimizing the product. Google understood this early, paying billions to be the default on Safari and Firefox — effectively buying the same advantage Bing gets from Microsoft's OS ownership.
