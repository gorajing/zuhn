Bryk's thesis: in 5 years, GPT-5-level AI agents will be everywhere—navigating the web, doing tasks, managing workflows. Every single one of them will need search. Exa is building the search engine for this agent era, not for humans.

The analogy to the internet era is clear: the most valuable companies weren't the websites—they were the infrastructure (AWS, Google, Stripe) that every website needed. The same logic applies to the agent era. The infrastructure layer (search, payments, authentication, memory) that every agent needs is likely to be more valuable and more durable than any individual agent. Founders should ask: what will every AI agent need?
