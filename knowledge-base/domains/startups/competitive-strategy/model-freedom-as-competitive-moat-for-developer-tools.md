Eric from Kilo Code explained that their competitive moat comes from three multiplied factors: speed of development, community engagement, and model freedom. Kilo Code is fully open-source/open-core and lets users log in with their Minimax, ZDOT AI, or other coding plans to access any model. They grew to 1.5 million users in under a year by encouraging community PRs and feature requests.

The deeper insight is about where moat migrates in the agentic era. As agents spawn sub-agents, the ability to dynamically select the right model for the right task becomes critical infrastructure. Kilo Code is positioning for a future where agents, not just humans, choose models — making model-agnostic routing a platform-level capability. This aligns with Open Router's view that deployment differentiation (speed, latency, regional optimization) matters as much as model selection itself.
