In the shade ball experiment, a single layer allows swimming with difficulty — balls move past each other like a fluid. But adding a second layer transforms the system: balls trap against each other and behave like a solid, providing dramatically more resistance.

This maps directly to competitive moats and organizational dynamics. A startup with one product line stays nimble — components can shift independently. But as layers accumulate (products, processes, integrations, dependencies), the system develops emergent rigidity. This is both a moat (competitors can't easily displace an entrenched multi-layer system) and a liability (the incumbent can't easily adapt). The lesson: deliberately choose which layers to add, because each one locks in the layers below it.
