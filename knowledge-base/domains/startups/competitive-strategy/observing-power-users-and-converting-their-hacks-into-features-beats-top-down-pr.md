Twitter's most enduring features were not designed in boardrooms. Hashtags were invented by users who organically started tagging tweets to organize conversations -- Jack Dorsey actually opposed them as 'too complicated for the average person.' Retweets formalized the copy-paste behavior users were already doing. @mentions codified how users were referencing each other. Lists emerged from users' need to organize who they followed.

Ev Williams' genius was observation over invention. His team monitored which user hacks had momentum and converted them into official platform features. This bottom-up approach produced features that had already been market-validated before a single line of code was written. The contrast with later CEOs is stark: Dick Costolo's top-down decisions to chase Instagram with filters, buy Vine reactively, and add photo features all failed to stick because they were responses to competitors rather than responses to user behavior.
