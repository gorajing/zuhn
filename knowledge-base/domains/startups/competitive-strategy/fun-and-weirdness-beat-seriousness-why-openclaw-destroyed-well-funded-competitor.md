When asked why OpenClaw won against the entire agentic AI startup landscape, Steinberger's answer was simple: 'Because they all take themselves too serious.' He wanted OpenClaw to be fun and weird — the lobster branding, the space-lobster-in-a-TARDIS lore, agents with personalities defined by soul.md files, the install process requiring git clone and build (no polished onboarding).

This wasn't strategic positioning — it was authentic. Steinberger described building OpenClaw as the most fun project he'd ever worked on, treating it like Factorio with infinite levels to unlock. The soul.md innovation — letting users infuse agents with personality — created diversity that made MoltBook fascinating rather than uniform. The principle echoes through tech history: authentic enthusiasm and irreverence often outperform polished but soulless corporate products, especially in developer-facing open-source tools.
