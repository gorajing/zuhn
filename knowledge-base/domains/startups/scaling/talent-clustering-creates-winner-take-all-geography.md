Graham identifies a powerful feedback loop in talent distribution: great programmers want to work with other great programmers. This creates a clustering dynamic where talent concentrates in a small number of hubs — potentially just one. The implication is that talent geography is not static; it's a compounding advantage that can shift.

This is significant because it means a region's tech dominance is not permanent. If another hub begins attracting the best talent, the network effect could rapidly drain existing hubs. The 50-year horizon Graham describes is already materializing as remote work and global mobility increase the liquidity of the programmer labor market.
