Service businesses (78% of businesses) have fundamental capacity limits — a coach can only take so many clients, an agency can only start so many projects per month — but prospects default to assuming unlimited capacity unless told otherwise. Explicitly surfacing real constraints ('I can take 4 more clients this quarter', 'I start one cohort per month, next one begins Friday') creates scarcity that is honest, defensible, and convertible. This avoids the ethical and reputation costs of manufactured urgency (fake countdown timers, phantom inventory) while still exploiting the psychological pull of scarcity. For larger businesses, rolling cohort starts serve the same function — breaking unlimited capacity into discrete, time-bounded enrollment windows. The principle generalizes: any real constraint that prospects can't see is a scarcity asset waiting to be surfaced.
