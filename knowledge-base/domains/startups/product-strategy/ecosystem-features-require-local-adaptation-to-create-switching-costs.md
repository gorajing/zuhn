Apple's competitive moat in Western markets relies heavily on ecosystem lock-in: Apple Pay, iMessage, Siri, Maps, and seamless device integration create high switching costs. But in India, none of these services function adequately. Apple Pay wasn't available, Siri didn't understand local languages well, and Apple Maps lacked the granularity needed for Indian navigation.

This exposes a critical insight about ecosystem strategies: switching costs are only real if the ecosystem delivers local value. When core ecosystem services don't function in a market, the 'walled garden' has no walls. Competitors who build locally-relevant ecosystems (like Xiaomi's Mi ecosystem tailored to Indian needs) can capture users without fighting against switching cost gravity.
