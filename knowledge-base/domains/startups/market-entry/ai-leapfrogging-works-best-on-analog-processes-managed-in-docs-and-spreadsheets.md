NFX identifies a specific pattern: GRC workflows already cost $47 billion in software spend, yet the actual work still happens in docs and spreadsheets. This gap between spending and actual workflow modernization represents the highest-leverage AI opportunity because the improvement is not incremental — it's a 100x leap from manual compilation to instant generation.

This pattern generalizes beyond GRC. Any industry where expensive software exists but practitioners still fall back to manual document-based processes is ripe for AI leapfrogging. The key insight is that the size of the existing software market validates demand, while the persistence of analog workflows proves that existing solutions are insufficient. Founders should look for this specific combination: large TAM + spreadsheet-based actual workflow.
