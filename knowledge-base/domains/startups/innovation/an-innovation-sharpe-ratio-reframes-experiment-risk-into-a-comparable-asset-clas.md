The Sharpe Ratio measures return per unit of risk, making it one of the most common tools for comparing investments that differ in both return and volatility. Applying the same logic to innovation—(expected upside × success probability) / (experiment cost × expected failure rate)—produces a number that can be compared to private equity returns or high-yield bond yields.

This is not about false precision. The value is rhetorical and structural: it forces both the innovation team and the CFO to be explicit about assumptions rather than arguing vaguely about 'potential.' It also sets a bar—if the innovation Sharpe Ratio is lower than the company's venture fund allocation, that's a signal to revisit the experiment design. The framework is most useful as a conversation scaffold rather than a hard decision criterion.
