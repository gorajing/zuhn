Giant viruses were visible under microscopes for decades but consistently misidentified as bacteria because the mental model 'viruses are tiny' was so deeply embedded that size alone triggered automatic misclassification. This is a general epistemological pattern: when your categories have implicit constraints (viruses = small, startups = tech, etc.), evidence that violates those constraints doesn't register as anomalous — it gets silently absorbed into the nearest 'acceptable' category. The fix isn't better observation tools but deliberately stress-testing the boundaries of your classification systems. Ask: what would I expect to see if my category definition is wrong?
