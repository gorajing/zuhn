Bootcamps and tutorials teach syntax — loops, variables, functions — which feels like learning to code. But the salary premium in software engineering comes from computer science fundamentals: algorithm design, system architecture, debugging complex interactions, and decomposing ambiguous problems. Syntax is to programming what vocabulary is to writing novels — necessary but nowhere near sufficient.

This is a general pattern in skill acquisition: surface-level fluency in a domain's 'language' creates premature confidence while the deep structural skills remain undeveloped. The gap between syntax knowledge and problem-solving ability explains why many bootcamp graduates struggle in interviews despite being able to write working code for tutorial exercises. The Dunning-Kruger effect is amplified when the easy-to-learn surface layer (syntax) is visible and the hard-to-learn deep layer (algorithmic thinking) is invisible.
