Bonanno describes the stress response as 'magnificent' — far more sophisticated than the simple fight-or-flight label suggests. When sensory signals match threat patterns, the amygdala and hypothalamus trigger a cascade: blood flow and glucose increase to extremities, pupils dilate, unnecessary functions go offline, and the brain shifts from reflective processing to automatic, survival-focused processing.

If the threat continues, the hypothalamus sends neurochemical messengers to the adrenal glands, releasing cortisol. Cortisol then bonds with brain structures causing epigenetic changes that amplify the response. Crucially, the system has a built-in brake: some cells have high cortisol affinity (responding immediately and powerfully) while others have low affinity (building up gradually then shutting the response down). This entire arc can last hours, during which long-term memory is blocked because narrative recall would be distracting. The system explains both resilience (when it works effectively) and PTSD vulnerability (when cortisol levels are low or the brake is too sensitive due to prior epigenetic changes).
