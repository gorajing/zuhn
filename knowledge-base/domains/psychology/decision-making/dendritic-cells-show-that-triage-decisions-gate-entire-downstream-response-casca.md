When the immune system faces an infection, the dendritic cell collects samples from enemies and makes a crucial binary decision: activate anti-virus forces or anti-bacteria forces. This single classification determines which entire cascade of T cells, B cells, and antibody types get deployed. Every downstream resource allocation flows from this one triage call.

This pattern generalizes beyond biology. In any complex system with multiple specialized response capabilities, the early classifier that routes between them is the most leveraged node. Getting the classification wrong doesn't just waste one resource — it activates the wrong entire branch while the correct branch sits idle. This is why triage functions (ER intake, incident severity classification, threat assessment) deserve disproportionate investment in accuracy relative to any single downstream capability.
