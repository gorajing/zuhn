Graham draws the analogy to physical strength: preindustrial work built strength by necessity; now only those who choose to exercise develop it. The population-level result is that most people are weaker than their ancestors — not because strength became impossible, but because the environmental forcing function disappeared.

This is a general pattern: skills that technology makes unnecessary become rare not because they're blocked, but because the default path no longer requires them. Writing is heading the same way. The implication for anyone who values clear thinking: the skill will require explicit, deliberate cultivation — like going to the gym — not merely showing up to work.
