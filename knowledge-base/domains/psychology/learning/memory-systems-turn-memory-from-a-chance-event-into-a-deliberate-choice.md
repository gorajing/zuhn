The pre-memory-system relationship to learning is: you read something, maybe remember some of it, probably forget most of it, and there's nothing to be done about it. This is memory as chance event. Memory systems (Anki, SuperMemo, Quizlet, Duolingo, the Matuschak-Nielsen mnemonic medium) invert the relationship: by following a few simple rules (card atomicity, spaced review intervals, minimal review time), users can guarantee that material stays in long-term memory for as long as they want. This is memory as choice. The change is not just practical (learning more stuff) but psychological — when you know you can retain anything you decide to retain, you read differently, you engage differently, you spend cognitive budget on higher-order questions instead of worrying about forgetting. Cognitive scientists have known the mechanics for decades; what was missing was a medium that operationalized the mechanics so most users actually do it.
