Creating fundamentally new things is so recent in human history that we haven't developed cultural norms for how to evaluate early versions of ambitious projects. We default to judging them either as finished work (too harsh) or as less ambitious projects (missing the point). This is a category error, not a character flaw — early ambitious work is genuinely a special case that requires a different evaluation polarity.

The correct response to early-stage ambitious work is not lower standards but reversed polarity: instead of listing reasons it won't work, actively searching for ways it could work. Silicon Valley's cultural innovation is precisely this polarity reversal, which is why it produces outsized outcomes despite also being occasionally fooled by impostors. The error rate from excessive optimism is far cheaper than the error rate from excessive skepticism when outcomes follow a power-law distribution.
