Paul Graham observes that when rankings must be created without meaningful criteria, the situation degenerates into a popularity contest — rank depends on one's ability to increase one's rank, not on any external measure of skill or contribution. This mirrors court dynamics where there is no external opponent, so members become each other's opponents.

This pattern extends far beyond schools: corporate teams without clear missions, online communities without shared projects, and social clubs without purpose all exhibit the same degeneracy. The antidote is ensuring groups have real external tests — shipping products, winning competitions, serving customers — so that hierarchy tracks competence rather than social manipulation. Leaders designing organizations should treat purposelessness as the root cause of toxic politics, not bad individuals.
