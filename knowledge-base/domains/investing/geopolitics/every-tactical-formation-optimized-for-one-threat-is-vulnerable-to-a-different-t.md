Napoleonic infantry had four formations: line (maximum firepower but fragile), column (momentum for assault but narrow frontage), square (impervious to cavalry but vulnerable to artillery), and skirmish (flexible but weak). Each was optimized against one threat but vulnerable to others. The key tactical skill was transitioning between formations as threats changed — which required training, discipline, and quick decision-making.

In business, organizational structures face the same tradeoffs: flat structures are agile but lack coordination, hierarchical structures coordinate well but are slow, matrix structures balance both but create confusion. Product-focused organizations ship fast but miss cross-product synergies. The key insight isn't finding the 'best' structure — it's building the capability to reorganize quickly as competitive threats change.
