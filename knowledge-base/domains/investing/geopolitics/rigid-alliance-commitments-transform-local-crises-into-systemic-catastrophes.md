The 1914 alliance system was a catastrophe machine: Austria's conflict with Serbia automatically activated Russia's Slavic solidarity, which activated Germany's alliance with Austria, which activated France's alliance with Russia, which activated Britain's guarantee to Belgium. Each commitment was individually rational, but the system as a whole meant that any crisis anywhere would cascade into total war. No individual decision-maker intended world war — the system produced it.

In financial markets, this is exactly how systemic risk operates: counterparty obligations, derivatives chains, and mutual guarantees create a web where one failure cascades through the entire system. The 2008 financial crisis, where Lehman's bankruptcy triggered AIG's collapse triggered a global credit freeze, followed the same escalation pattern. The lesson: when evaluating risk, the question isn't 'what's the probability of this individual event?' but 'what system of obligations will this event trigger?'
