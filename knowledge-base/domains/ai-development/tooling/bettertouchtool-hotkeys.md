BetterTouchTool enables custom double-tap hotkeys that dramatically
speed up app switching in Claude Code workflows. Configure CMD+CMD to
switch to Claude Code and OPT+OPT to switch to the browser — faster
than CMD+Tab cycling through windows. It also supports a relative URL
copy shortcut from Cursor, which is useful for sharing file paths with
Claude. These small friction reductions compound over a full day of
switching between Claude, browser, and editor. The double-tap pattern
is intuitive and does not conflict with existing system shortcuts.
