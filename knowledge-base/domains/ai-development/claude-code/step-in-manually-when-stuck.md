Not every task is best suited for Claude. When Claude gets stuck in a
loop — repeatedly trying and failing to fix something, generating
increasingly convoluted solutions — recognize when human intervention
is more efficient. The 30-minute rule is a good heuristic: if Claude
has been struggling for 30 minutes on something you know you could fix
in 2 minutes, step in. Common cases include environment-specific issues,
subtle CSS tweaks, config file quirks, and problems requiring visual
verification. Knowing when to hand-code is as important as knowing when
to delegate to Claude.
