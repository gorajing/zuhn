Ng observes that early LLMs were optimized to answer human-generated questions — queries like 'why did Shakespeare write Macbeth.' But agentic workflows require different capabilities: reliable function calling, structured output for tool use, and the ability to serve as components in iterative pipelines rather than standalone conversational agents.

The trend toward models explicitly tuned for tool use (and Anthropic's computer use capabilities, which Ng references) represents a fundamental shift in what LLMs are being optimized for. This creates a 'lift' that raises the ceiling for agentic workloads. For AI system builders, this means the gap between what's possible with agentic workflows today and what will be possible with purpose-tuned agentic models is substantial — the current impressive results are achieved despite models being primarily tuned for a different task.
