Many agents optimize for speed-to-outcome — generating an output quickly — but the harder and more valuable target is speed-to-understanding: grasping the user's nuances, implicit constraints, and preferred approach so the output matches what they actually wanted in the way they wanted it. An output that ignores those nuances is useless no matter how fast it arrives; the distinction is doing 'the right thing' versus doing 'just something.' Personalization is the lever: give the agent the thoughts, systems, knowledge, principles, and patterns you would have applied yourself. Harvey's playbooks encode a legal firm's own methods and principles for tasks like contract review so the agent works the firm's way; Harvey also uses persistent memory created during interactions and drawn on later. Claude offers skills, connectors, and systems you connect to grow the agent's knowledge base and improve personalization.
