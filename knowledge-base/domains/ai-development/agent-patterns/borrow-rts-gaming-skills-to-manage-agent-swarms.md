The worry that we won't adapt to a world of 'telling agents to code for us' assumes the required skills are brand new. Salomon's reframe is that they aren't — they've been 'hiding in unexpected places,' chiefly real-time strategy games where players already manage dozens of units, cycle attention with muscle memory, and read battlefield state at a glance. Agent Craft deliberately ports these mechanics: hotkey-style cycling between agents that need approval, heat maps, spatial unit selection. The cross-domain lesson is broader than one product — when a new workflow feels overwhelming, look for an existing domain where humans already perform the same cognitive task fluently, and import its interface conventions. The transfer lowers the learning curve because operators reuse trained reflexes instead of building new ones.
