Casas opens with the inflection: by late 2025, models produced high-fidelity, accessible, animated UI from a single prompt — a one-line 'rewrite my blog' yielded a blurred, accessible search box he never asked for. Yet the dominant paradigm is still mostly static UI, and the 'Jarvis moment' of floating, ephemeral windows hasn't arrived.

Borrowing Karpathy's framing, we have a new computer whose GUI hasn't been invented — like the 1970s, super-intelligence with no mature interface language, so everyone defaults to chat (SaaS companies bolting chat onto homepages). This is a capability overhang: the generative power exists but the interaction grammar to channel it doesn't. For product builders, the strategic opening isn't training a better UI model — it's discovering the interface paradigm that lets existing capability express itself beyond chat boxes and static components.
