Zitron's Hummer analogy is precise: if you never pay for gas, you'll drive a gas-guzzler everywhere and never think about efficiency. Claude Code subscribers operate in exactly this illusory environment. On a $20/month subscription, users may burn $150+ in actual compute. On $200/month, over $2,700.

The critical psychological insight: users currently forgive hallucinations, circular reasoning, and wasted tokens because it 'feels free.' If moved to API pricing where every mistake costs real money — every hallucination, every misinterpretation, every security setting left misconfigured — the tolerance would evaporate. This means all current enthusiasm metrics (user satisfaction, adoption stories, productivity claims) are contaminated by the subsidy. Anthropic cannot raise prices 5-7x without destroying usage, but cannot sustain current pricing without infinite fundraising. There is no middle path.
