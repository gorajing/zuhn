A year before, three senior software engineers armed with ChatGPT hit the persistence problem — settings wouldn't survive a reboot — and abandoned the phone for a year. Claude reached the exact same dead end but, instead of stopping, proposed spinning up a Windows VM, ran the proprietary software inside it, and built a TCP proxy to intercept and decode the real protocol. Starkov, a self-described 'normal software engineer' with zero security background, stresses the agent made the project not 10x faster but possible at all: 'it was just smarter than me.' This is a concrete data point on capability thresholds — the difference between model generations is not incremental speedup but a binary flip from 'expert-only / impossible' to 'achievable by a generalist.' The time-sensitivity is the point: the bar moves, and tasks that are infeasible today may flip with the next model.
