Ng presents the AI stack as four layers: semiconductors, cloud infrastructure, foundation models, and applications. While media hype concentrates on the technology layers (new models, new chips), he argues the application layer must work out 'even better' by economic necessity — applications need to generate enough revenue and value to pay all the technology providers below them.

This is a structural argument, not a prediction. Just as the internet's application layer (Google, Amazon, Facebook) eventually dwarfed the infrastructure layer (ISPs, hosting), AI applications will ultimately capture more value than the models and compute they run on. The emerging agentic orchestration layer (LangChain, LangGraph, and similar frameworks) is creating new opportunities by making it easier for developers to build sophisticated applications on top of foundation models. For builders, the implication is clear: the biggest returns will come from solving real-world problems with AI applications, not from building foundation models.
