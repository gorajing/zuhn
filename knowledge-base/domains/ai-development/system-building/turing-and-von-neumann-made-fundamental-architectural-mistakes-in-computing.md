Modern computing separates program (software) from material (hardware). Gershenfeld argues this is a fundamental error that biology avoided: DNA is simultaneously the description and part of the material. The ribosome reads instructions that are physically part of the system it builds.

This separation created enormous engineering challenges that biological systems don't face. The future of computing may involve merging digital instructions with physical material - programmable matter that IS its own program - rather than continuing to run software on inert hardware.
