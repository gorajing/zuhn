Smith highlights a critical progression: currently, human engineers still review AI-generated code, but as AI code quality improves, even this review step becomes uneconomical. The Spotify example — where top developers no longer write code — and the CNBC journalists cloning an app with zero experience illustrate how fast this transition is happening.

The implication for software professionals is that the current 'human-in-the-loop' phase is transitional, not a stable equilibrium. The METR curve showing AI completing increasingly long software tasks suggests the oversight window is narrowing. Engineers who position themselves as AI orchestrators rather than code writers have a longer runway, but even that role faces compression as AI agents become capable of longer autonomous work sequences.
