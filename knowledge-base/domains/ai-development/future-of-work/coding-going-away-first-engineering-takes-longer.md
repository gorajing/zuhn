Dario Amodei distinguishes between "coding" (writing code) and "software
engineering" (architecture, design, understanding user needs, managing
teams). AI models are handling the raw coding part first — Anthropic sees
this internally with Claude Code used by their own engineers daily. But
the broader software engineering discipline, which requires understanding
context, making architectural decisions, and coordinating human systems,
takes longer to automate. Eventually, end-to-end software engineering
will be AI-driven too, but there's a meaningful window where the
engineering skills around code remain valuable even as the coding itself
becomes commoditized.
