Jeff from Hugging Face described a concrete example of skill democratization: Hugging Face Skills tells agents how to use the TRL library for reinforcement learning with GRPO on custom datasets. Two years ago, this required a qualified data scientist working inside a company. Now anyone can instruct an agent to do it.

This represents a pattern where each generation of AI tooling compresses the previous generation's expert-level tasks into commodity operations. The 'video game levels' metaphor Jeff used is instructive: level one is discovering and downloading models, level three is fine-tuning on custom data. As agents absorb the knowledge to execute higher levels, the entry barrier drops and the frontier of what constitutes 'expert work' shifts upward. For builders, this means the window of competitive advantage from any single technical capability is shrinking — moats must come from problem understanding and customer obsession, not technical execution.
