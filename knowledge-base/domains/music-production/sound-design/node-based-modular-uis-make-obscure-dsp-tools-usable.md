Composer's Desktop Project has contained Aphex-grade processing since the 90s, but its command-line interface kept it in the hands of a tiny academic cohort. SoundThread puts a modular patch graph on top — drag wires, right-click to browse processes, hit run — and suddenly the same power is accessible to anyone who understands a Eurorack. The lesson generalizes: when a capability exists but adoption is gated by interface friction, a thin visual wrapper can 100x its reach without changing a line of the underlying engine. For producers, this means auditing forgotten free tools whenever a new UI appears.
