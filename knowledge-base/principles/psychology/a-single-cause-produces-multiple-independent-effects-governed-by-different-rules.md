A gunshot underwater creates three physically independent phenomena each described by different equations; photosynthesis runs as two decoupled stages optimizable separately; Bernoulli's principle unifies cavitation across bullets, propellers, pipes, and arteries. When one input fans out into multiple outputs, each branch obeys its own logic. Treating a multi-effect system as one thing guarantees you will miss mechanisms, overfit to one branch, or import constraints from one domain that do not apply to another. The correct first move is always decomposition: identify which independent effects a single cause produces, then analyze each on its own terms.
