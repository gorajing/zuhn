Political visibility creates a separating equilibrium where any lean gets amplified into full tribal membership. When tribal identity activates, people reject valid arguments purely based on source. Groups persecute outsiders not from hatred but because shared enemies create the strongest internal bonds. Group identities persist because their arbitrary elements create the cohesion needed to carry beneficial beliefs.
