Asymptotic network effects plateau in value quickly, making companies with this structure vulnerable to well-funded competitors who only need to match a fraction of the supply. When users can freely use competing marketplaces simultaneously, network effects erode entirely, making supply-side lock-in the critical strategic priority. DoorDash's value comes from its network position, not its code -- but that network has a ceiling.
