Competitive outcomes are decided by accumulated scale, not superior design. The 'barely aesthetic' x86 survived while elegant RISC architectures failed because install base trumps design purity. Data network effects are invisible at small scale but decisive at planetary scale -- 4 customers vs 3 means nothing, but 4 billion vs 1 billion is game over. Linux dominates because its open ecosystem covers more applications than any proprietary system. In AI, owning the full workflow generates proprietary data that public models cannot replicate, creating compounding advantages that grow with every customer.
