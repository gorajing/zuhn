Stripe's minions achieve high one-shot success rates specifically because Stripe has invested heavily in developer productivity tooling, documentation, and blessed paths for common workflows. When an agent needs to add a new API field, there's clear documentation on exactly how to do it — the same docs that help human engineers help the agent succeed. The insight is bidirectional: what's good for the developer is good for the agent, and vice versa. This means DX investment now has a multiplied ROI — it accelerates both human and agent engineering. For engineering leaders trying to justify DX investment, attaching it to an AI initiative is described as 'the secret way' to get budget for developer experience improvements.
