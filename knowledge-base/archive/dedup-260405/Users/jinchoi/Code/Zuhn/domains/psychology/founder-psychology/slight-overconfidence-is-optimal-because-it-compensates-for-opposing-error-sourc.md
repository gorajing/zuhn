Conventional wisdom says perfectly calibrated self-assessment is optimal, but Graham argues this ignores that self-assessment operates in a system with multiple error sources pushing in the negative direction — other people's skepticism, your own fear of producing bad work, and the inherent difficulty of judging unfinished projects. Slight overconfidence compensates for these opposing errors, much like how overestimating a project's importance compensates for underestimating early results.

This has a precise mathematical analogy: if you're 20% done on something worth 100 but judge it as 10% done, overestimating the goal's value to 200 gives you the correct expected value despite both components being wrong. The implication is that perfect calibration is only optimal in a frictionless environment — in the real world of social and psychological headwinds, a small positive bias is the rational adjustment.
