The biggest obstacle to ambitious work is often the first step. Framing a project as a sketch, quick hack, or experiment achieves two things: it lowers the standard you judge early results against, and it reduces the emotional commitment that makes abandonment feel like failure. Once the project gains momentum, you can sneakily convert it into something more ambitious.

This pairs with choosing media that support fast iteration and low upfront commitment. Digital tools beat physical ones partly because starting a new file costs nothing. Game modding flourished when platforms like Minecraft made the aesthetic deliberately crude, so gameplay innovation became the primary value rather than production quality. The pattern: reduce the gap between early work and 'acceptable' work, and more people will start — and finish — ambitious projects.
